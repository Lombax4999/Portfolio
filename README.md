# Matthew Frommeyer Portfolio

## Files

- `index.html` — page structure and résumé content
- `styles.css` — layout, responsive styling, light/dark themes, and animations
- `script.js` — terminal intro, theme toggle, mobile navigation, and scroll reveals
- `Matthew_Frommeyer_Resume.pdf` — résumé linked from the website

## Open the site

Double-click `index.html` to preview it locally in a browser.

For the best results, serve the folder with a simple local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Before publishing

1. Replace the placeholder LinkedIn note with your actual LinkedIn URL.
2. Add a GitHub link if you want one displayed.
3. Review the project descriptions and adjust any details you want to make more specific.
4. Upload the entire folder to GitHub Pages, Netlify, or another web host.
